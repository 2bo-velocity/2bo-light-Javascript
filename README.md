# 2bo Light Javascript

A lightweight, dependency-free, capsule-style Javascript micro library.

### Features

Ultra-lightweight DOM utility layer for modern browsers.

- Global `$2b` object.
- DOM manipulation (`q`, `qa`, `id`).
- Style manipulation (`addClass`, `removeClass`, `css`).
- Attribute manipulation (`attr`).
- Event handling (`on`, `off`).
- Utilities (`ready`, `defer`, `uuid`).
- Ajax (`get`, `post`).

## Philosophy

2bo Light Javascript is designed to stay:

- Small
- Stable
- Predictable
- Dependency-free

This library will never:

- Extend native prototypes
- Become a framework
- Add build step requirements
- Support legacy browsers
- Add unnecessary features

## Browser Support

Supports only the latest stable versions at release time:

- Edge (Chromium)
- Firefox
- Chrome
- Safari

No legacy support.
No polyfills included.
Forward-only compatibility model.

## Installation

### Direct Script

```html
<script src="dist/2boLight.min.js"></script>
```
