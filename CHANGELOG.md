# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-14

### Added
- Initial release of 2bo Light Javascript.
- Core features implemented:
  - Global `$2b` object.
  - DOM manipulation (`q`, `qa`, `id`).
  - Style manipulation (`addClass`, `removeClass`, `css`).
  - Attribute manipulation (`attr`).
  - Event handling (`on`, `off`).
  - Utilities (`ready`, `defer`, `uuid`).
  - Ajax (`get`, `post`).
